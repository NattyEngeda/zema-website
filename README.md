## Build and Deploy

Install Composer 

```bash
  composer install
```

Generate Keys

``` baseh 
    php artisan key:generate
``` 

Migrate Database 

```bash 
    php artisan migrate 
```

Serve Project 

```bash 
    php artisan serve 
```
